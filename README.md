# 🌫️ AQI Prediction System — India Air Pollution Dashboard

An interactive dashboard that analyzes historical air quality data across Indian cities and predicts AQI using machine learning, with a live, real-time tab that pulls current air quality data and a future forecast for any city in India.

## 📌 About the Project
Air pollution is one of India's biggest public health concerns, but most people only get a vague sense of "how bad" the air is on a given day. This project digs into historical air quality data from 26 Indian cities (2015–2020), calculates AQI using the official CPCB breakpoint method, and trains several machine learning models to classify and predict it. On top of that, it connects to a live air quality API so you can check the real-time AQI of almost any city in India right now, along with a short-term forecast generated by our trained model.

## ✨ Features
- 🏠 **Overview** — quick stats: total records, number of cities, years covered, average AQI, and model accuracy
- 📊 **EDA** — AQI distribution, monthly and seasonal trends, pollutant correlation heatmap
- 🏙️ **City Analysis** — city-wise AQI ranking, category breakdown, and pollutant heatmaps
- 🤖 **ML Models** — compares Random Forest, Logistic Regression, and Decision Tree classifiers, plus two regression models, with confusion matrix, feature importance, and 5-fold cross-validation
- 🔮 **Predictor** — manually enter pollutant values and get an instant AQI prediction
- 🌍 **Live & Forecast** — real-time AQI for any Indian city (via the WAQI API) plus a multi-day AQI forecast from our trained model
- 📋 **Raw Data** — explore, filter, and download the underlying dataset

## 🧠 How AQI is Calculated
We use the official CPCB (Central Pollution Control Board) breakpoint formula. Each pollutant — PM2.5, PM10, NO2, SO2, CO, and O3 — gets its own sub-index, and the overall AQI is the **maximum** of these sub-indices. This is the same convention followed by CPCB and the US EPA.

## 🛠️ Tech Stack
- Python
- Streamlit (dashboard/UI)
- Pandas & NumPy (data processing)
- Scikit-learn (ML models)
- Plotly (interactive charts)
- WAQI API (live air quality data)

## 📂 Dataset
`city_day.csv` — day-wise historical air quality readings from CPCB monitoring stations across 26 Indian cities (Ahmedabad, Aizawl, Amaravati, Amritsar, Bengaluru, and more) from 2015 to 2020.

## ⚙️ Run It Locally
```bash
git clone <this-repo-url>
cd AQI-Prediction-System
pip install -r requirements.txt
streamlit run app.py
```
To enable the Live & Forecast tab, add your free WAQI token in Streamlit secrets (get one at aqicn.org/data-platform/token).
